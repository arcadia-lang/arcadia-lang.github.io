# Replies

## **Reply Words**

Arcadia has **three primary reply words**, each serving a distinct conversational function:

- **si** → **"Yes"** (affirmative response).
- **ni** → **"No"** (negative response).
- **ack** → **Neutral acknowledgment**—used to show that the listener is paying attention or understands **without confirming or denying** the statement.

## **Generic Reply Words**

Similar to English words like **"nowhere," "somewhere," "anywhere," and "everywhere,"** Arcadia has **structured generic reply words** that align with **interrogatives**.

### **Interrogative-Based Reply Forms**

Each interrogative combines with **prefixes** —after dropping the initial "c"— to generate **negative, generic, specific, and collective** responses.

- Prefix System:
    - **in-** → Negative form (absence or negation).
    - **ej-** → Unknown generic (any).
    - **aliam-** → Unknown specific (some).
    - **ol-** → Collective reference (all).

| Category               | cuídh     | cuál     | cubi     | cuandho     | cuót     | cume     |
| ---------------------- | --------- | -------- | -------- | ----------- | -------- | -------- |
| **Negative**           | inuídh    | inuál    | inubi    | inuandho    | inuót    | inume    |
| **Unknown (generic)**  | ejuídh    | ejuál    | ejubi    | ejuandho    | ejuót    | ejume    |
| **Unknown (specific)** | aliamuídh | aliamuál | aliamubi | aliamuandho | aliamuót | aliamume |
| **Collective**         | oluídh    | oluál    | olubi    | oluandho    | oluót    | olume    |
