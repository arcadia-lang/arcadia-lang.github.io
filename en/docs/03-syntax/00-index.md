# Syntax

Arcadian syntax is verb-first and role-based.
The verb functions as the structural core of the clause: it licenses participant roles and determines which roles are mandatory or optional.

Arcadia does not categorise verbs by transitivity.
Valency emerges from verb type and role structure, not from a transitive/intransitive distinction.

## Clause Structure

The default clause order is:

**Verb – Core Roles – Obliques**

Core roles and obliques may be reordered for discourse emphasis, poetic rhythm, or stylistic effect.

Role assignment is never determined by word order.
Roles must remain recoverable through morphology and verb type.

Arcadia is fully pro-drop.
Core roles may be omitted when recoverable from verbal morphology or discourse context.

## Roles

Each verb type defines a set of participant roles.
Some roles are mandatory, while others are optional.

Arcadia is pivotless.
No participant is grammatically privileged over the others, and no role is promoted to control the clause.
Participant roles are identified directly through semantic case marking.

In all verb types, the verb comes first in the sentence structure and is followed by its assigned roles.
Core roles follow in the canonical order defined by the verb type, first the mandatory ones, and finally the optional ones.
Case marking allows roles to be reordered when the intended structure remains recoverable.

## Verb Types

Verb types and their role structures are defined in [Verb Types][verb-types].

The current system distinguishes four core verb families:

- existential verbs;
- relational verbs;
- mental verbs;
- agentive verbs.

These families may contain more specific subtypes.
