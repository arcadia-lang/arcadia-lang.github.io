# Numerals

Arcadia uses **two parallel number systems**:

- **Decimal system** → Commonly used for general counting and calculations.
- **Hexadecimal system** → Included for **future-proofing**, allowing expanded numerical representation.

### **Numbers and Writing**

- **Hexadecimal notation uses `0-9` and `A-F`**, ensuring **no overlap with written words**, since Arcadia uses only **lowercase letters** for textual representation.
- **Numbers are grouped using underscores (`_`)** for clarity and **unambiguous readability**.

---

## **Larger Numbers**

- After **999_999**, both systems use **powers of 1_000_000** to create **names for arbitrarily large numbers**.
- For **one-word power formations**, Arcadia **replaces "ad X potente"** with the **suffix "-ente"**.

### **Decimal Power Names**

| **Decimal**       | **Name**                                     |
| ----------------- | -------------------------------------------- |
| 1_000_000         | **milonu**                                   |
| 1_000_000_000_000 | **dosente milonu** _(milonu ad dos potente)_ |

### **Hexadecimal Power Names**

| **Hexadecimal**     | **Name**                               |
| ------------------- | -------------------------------------- |
| 0x1_000_000         | **bet**                                |
| 0x1_000_000_000_000 | **dosente bet** _(bet ad dos potente)_ |
