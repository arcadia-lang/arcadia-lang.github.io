# Determiners

Determiners are words that precede a noun, providing additional information about **its reference, quantity, or specificity**.

Arcadia recognizes **six types** of determiners:

- **[Articles][article]** → Defines specificity: _the_ tree, _an_ apple.
- **[Demonstrative determiners][demonstrative determiner]** → Indicates proximity: _this_ machine, _that_ person.
- **[Interrogative determiners][interrogative determiner]** → Forms questions: _which_ book, _what_ blanket.
- **[Possessives][possessive]** → Expresses ownership: _Maria's_ bag, _my_ bed.
- **[Quantifiers][quantifier]** → Describes quantity: _a few_ dogs, _each_ child.
- **[Numerals][numeral]** → Specifies an exact number: _two_ dolphins, _a million_ dollars.

- [x] **Determiners can be combined**, as in **"the three musketeers."**

### **Determiner Rules in Arcadia**

- [x] **A determiner is mandatory** unless the noun is in the **zero**, or **collective number**, where **no determiner normally used**.
- [x] **Proper nouns** do **not** require a determiner but **can take one** when specificity needs to be emphasized.
