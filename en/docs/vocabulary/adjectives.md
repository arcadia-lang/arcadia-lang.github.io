# Adjectives

In Arcadia, adjectives match the **class of the noun they modify**, and thus exist in all five classes.
However, in dictionaries, adjectives are always listed in **-i class**.

# Opinion

| ------- | --------- |
| beli    | beautiful |
| faedhi  | ugly      |
| boni    | good      |
| mali    | bad       |
