# Orthography of Arcadia

Arcadia uses **23 letters** from the Latin alphabet, **excluding k, q, and w**.
Each letter has a **fixed pronunciation**, ensuring a **consistent spelling-to-sound correspondence**.

## Digraphs

Arcadia also uses the following **digraphs**:

| Digraph | IPA  | English Approximation |
| ------- | ---- | --------------------- |
| ch      | /t͜s/ | **ch**air             |
| dh      | /ð/  | **th**ere             |
| th      | /θ/  | **th**anks            |

!!! tip "Rule"

    The phoneme **/ks/** is always spelled **"x"**, never as "cs."

!!! warning "Pronunciation"

    Each letter is pronounced according to its **IPA value** and may differ slightly from English pronunciation.
    The **English equivalent** provides only the closest approximation (based on Received Pronunciation).

## Accentuation

Arcadia uses the **acute accent (´)** to mark stress:

- Words **without an acute** → Default stress on the **penultimate syllable** (or the **only syllable** if the word is monosyllabic).
- Words **with an acute** → Stress falls on the **marked syllable**.

Examples:

- **homis** (_human, person_) → stress on "o" (**ho**mis).
- **ni** (_no_) → stress on "i" (**ni**).
- **asé** (_third-person pronoun_) → stress on "é" (as**é**).

## Vowel Merging & Macron Notation

When **compounding words** or **adding affixes**, **identical adjacent vowels** are **never allowed**—they always **merge into a single instance**.

!!! tip "Rule"

    In **dictionary entries, grammar books, and syntax guides**, vowel mergers can be marked with a **macron** over the vowel (e.g., _mē_ instead of _me_).
    However, **macrons are not used in normal writing**—words are spelled without them in everyday text.

## Punctuation & Formatting

Arcadia has **no capitalization** and **no silent letters**.
It uses the following **punctuation marks**:

| Mark | Name             | Usage                                                                      |
| ---- | ---------------- | -------------------------------------------------------------------------- |
| .    | Period           | Ends a declarative sentence.                                               |
| ;    | Semicolon        | Connects related independent clauses or separates items in a complex list. |
| ,    | Comma            | Separates elements in a list or clauses within a sentence.                 |
| :    | Colon            | Introduces a list, quotation, or explanation.                              |
| !    | Exclamation Mark | Expresses strong emotion or a command.                                     |
| ?    | Question Mark    | Marks a question.                                                          |
| ""   | Quotation Marks  | Encloses direct speech, quotations, or titles.                             |

### **Hyphens & Apostrophes**

- **Hyphens (-)** → Used for compound words forming a single lexeme.
- **Apostrophes (’)** → Used for contractions.
- **Single Quotes (')** → Can substitute apostrophes for easier typing.

!!! note "Punctuation & Quotes"

    Punctuation should be placed **outside** quotation marks **unless** both the quoted sentence and the main sentence **end simultaneously**.

## Proper Nouns & Loanwords

- Proper names **retain their original spelling** (if in Latin alphabet).
- Names from other scripts **use standard Latin transliteration**, or a phonetic adaptation.
- Even with original spelling, names are pronounced **as close as possible** to the source language.
- As a **reader's aid**, a name can be **transliterated inside double colons** the first time it appears:
    - Example: "`mike::maic::`"
- Loanwords may contain the letters k, q, or w, even though they do not appear in native Arcadian words.
