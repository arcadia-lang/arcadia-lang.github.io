# Orthography and Phonology

This section defines the native writing system, punctuation, sound inventory, stress system, and phonotactic rules of Arcadia.

Arcadian spelling is designed to be transparent: native graphemes correspond predictably to phonemes, and phonological alternations are reflected consistently in writing.
The orthography is lowercase by default, avoids silent letters, and reserves non-native spellings for names, loanwords, symbols, and technical notation.

## Core Principles

Arcadian phonology follows five main principles:

1. **Small vowel system**: Arcadia has five vowel phonemes: /i e a o u/.
2. **Predictable vowel length**: the stressed vowel is phonetically long; vowel length is not independently contrastive.
3. **Symmetric voicing pairs**: most obstruents occur in voiced and voiceless pairs, supporting regular grammatical alternations.
4. **Restricted marked sounds**: /t͡ʃ/ and /d͡ʒ/ are not ordinary inherited root consonants; they primarily mark dubitative morphology and occur in loanwords.
5. **Soft but ancient sound profile**: the language favors sonorants, controlled fricatives, dental endings, and limited clusters while avoiding excessive harshness.

## Notation

| Notation | Meaning                                         |
| -------- | ----------------------------------------------- |
| / /      | Phonemic transcription.                         |
| [ ]      | Phonetic transcription.                         |
| ⟨ ⟩      | Orthographic form or grapheme.                  |
| C        | Any consonant.                                  |
| V        | Any vowel.                                      |
| R        | A liquid: /l/, /ʎ/ or /ɾ/.                      |
| N        | A nasal: /m/, /n/, or /ɲ/.                      |
| O        | An obstruent: plosive, fricative, or affricate. |

Unless otherwise specified, phonological rules apply to native Arcadian vocabulary only.
