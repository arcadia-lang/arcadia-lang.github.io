# Letters

This file defines Arcadian letters, digraphs, and their phonemic values.

## Vowels

Arcadia has five vowel phonemes:

```ipa
/i e a o u/
```

Each vowel can occur in stressed and unstressed syllables.
Vowel quality remains stable across stress positions, though the stressed vowel is phonetically long.

| Height | Front | Central | Back |
| ------ | ----- | ------- | ---- |
| Close  | /i/   |         | /u/  |
| Mid    | /e/   |         | /o/  |
| Open   |       | /a/     |      |

## Sonorants

Sonorants do not participate in secondary devoicing.

| Class       | Phonemes | Graphemes |
| ----------- | -------- | --------- |
| Nasals      | /m n ɲ/  | ⟨m n ny⟩  |
| Liquids     | /l ɾ ʎ/  | ⟨l r ly⟩  |
| Approximant | /j/      | ⟨j⟩       |

## Marked Affricates

The affricates /t͡ʃ/ and /d͡ʒ/ are not ordinary inherited root consonants.

In native grammar, /d͡ʒ/ is associated with the dubitative mood, and /t͡ʃ/ is its secondary-clause devoiced counterpart.

Outside this grammatical function, ⟨tx⟩ and ⟨dx⟩ are permitted mainly in loanwords, names, or explicit phonetic transcription.

## Gemination

Gemination is represented orthographically by doubling.

For single-letter graphemes, the consonant letter is doubled.

For digraphs, only the final letter is doubled:

| Base Type               | Gemination Rule             |
| ----------------------- | --------------------------- |
| single-letter consonant | double the consonant letter |
| digraph ending in ⟨h⟩   | double ⟨h⟩                  |
| digraph ending in ⟨x⟩   | double ⟨x⟩                  |
| digraph ending in ⟨y⟩   | double ⟨y⟩                  |

Gemination is phonemic where specified by the lexicon or morphology.
