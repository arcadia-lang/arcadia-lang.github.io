# Impersonal Pronouns

Arcadia has **two impersonal pronouns**, **"le"** and **"alé"**, which differ in **degree of specificity**.

- [x] **"le"** → Represents an **implied but relatively specific** vague subject (e.g., a known general group).
- [x] **"alé"** → Represents a **fully indeterminate** or **diffuse** subject (e.g., society as a whole).

---

## **Impersonal Pronoun Declensions**

Both **"le"** and **"alé"** follow standard noun declension patterns.

### **_le_ (Specific Impersonal)**

| **Case/Number**  | **Singular** | **Fractional** | **Plural** |
| ---------------- | ------------ | -------------- | ---------- |
| **Nominative**   | leis         | ledis          | lesis      |
| **Accusative**   | leon         | ledon          | leson      |
| **Dative**       | leim         | ledim          | lesim      |
| **Ablative**     | lea          | leda           | lesa       |
| **Predicative**  | lei          | ledi           | lesi       |
| **Adpositional** | le           | lede           | lese       |
| **Vocative**     | leo          | ledo           | leso       |

---

### **_alé_ (Vague Indeterminate Impersonal)**

| **Case/Number**  | **Singular** | **Fractional** | **Plural** |
| ---------------- | ------------ | -------------- | ---------- |
| **Nominative**   | aleis        | aledis         | alesis     |
| **Accusative**   | aleon        | aledon         | aleson     |
| **Dative**       | aleim        | aledim         | alesim     |
| **Ablative**     | alea         | aleda          | alesa      |
| **Predicative**  | alei         | aledi          | alesi      |
| **Adpositional** | alé          | alede          | alese      |
| **Vocative**     | aleo         | aledo          | aleso      |
