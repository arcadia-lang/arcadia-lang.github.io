# Gender

Arcadia is **not a gendered language**.
Gender has **no effect** on grammatical rules—it is **purely semantic**, used only when relevant to specify the gender of an object or person.

Gender is marked by the prefixes **"vi-"** (male) and **"me-"** (female).
If gender **does not need to be specified**, the noun remains **unmarked**.

!!! note "Using Gender"

    By default, gender **should not be marked** unless it is **explicitly relevant**.
    For example, when referring to a child, **"filio"** is used unless specifying **"son"** (*vifilio*) or **"daughter"** (*mefilio*).

## **Examples**

- **Parent**

    - **Any gender:** pareno
    - **Father:** vipareno
    - **Mother:** mepareno

- **Child (offspring)**

    - **Any gender:** filio
    - **Son:** vifilio
    - **Daughter:** mefilio
