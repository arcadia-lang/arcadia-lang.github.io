## **Τροποποιητές**

Η Αρκάδια περιλαμβάνει **δύο επίθετα** για την **σκιά**:

| Αρκάδια | Ελληνικά |
| ------- | -------- |
| lumo    | ανοιχτό  |
| scuro   | σκούρο   |

και **τέσσερα επίθετα** για το **φινίρισμα**:

| Αρκάδια   | Ελληνικά   |
| --------- | ---------- |
| metallico | μεταλλικό  |
| matto     | ματ        |
| gloso     | γυαλιστερό |
| neono     | νέον       |

Η σκιά πάντα **προηγείται** του φινιρίσματος.

## **Παραδείγματα**

1. _azulis glosis_

    - **translation:** glossy blue

2. _sangis scuris_

    - **translation:** dark red

3. _verdis lumis mattis_

    - **translation:** light matte green
