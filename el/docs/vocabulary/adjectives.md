# Επίθετα

Στην Αρκάδια, τα επίθετα έχουν την **τάξη του ουσιαστικού που τροποποιούν**, και έτσι υπάρχουν και στις πέντε τάξεις.
Ωστόσο, στα λεξικά, τα επίθετα αναφέρονται πάντα στην **στερεή τάξη**.
