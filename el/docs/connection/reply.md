# Απαντήσεις

## **Λέξεις απάντησης**

Η Αρκάδια έχει **τρεις κύριες λέξεις απάντησης**, καθεμία από τις οποίες εξυπηρετεί μια ξεχωριστή λειτουργία στην συνομιλία:

- **si** → **"Ναι"** (Καταφατική απάντηση).
- **ni** → **"Όχι"** (Αρνητική απάντηση).
- **ack** → **Ουδέτερη αναγνώριση**—Χρησιμοποιείται για να δείξει ότι ο ακροατής προσέχει ή κατανοεί **χωρίς να επιβεβαιώνει ή να αρνείται** την δήλωση.

- [x] **Χρήση του "ack"** → Χρησιμοποιείται συχνά σε **διάλογο** όταν κάποιος θέλει να **δηλώσει κατανόηση** αλλά να αποφύγει να δεσμευτεί σε μια οριστική θέση.

---

## **Γενικές λέξεις απάντησης**

Παρόμοια με ελληνικές λέξεις όπως **"πουθενά," "κάπου," "οπουδήποτε," και "παντού,"** η Αρκάδια έχει **δομημένες γενικές λέξεις απάντησης** που ευθυγραμμίζονται με **ερωτηματικά**.

### **Μορφές απάντησης με βάση το ερωτηματικό**

Κάθε ερωτηματικό συνδυάζεται με **πρόθετα** για να δημιουργήσει **αρνητικές, γενικές, ειδικές και συλλογικές** απαντήσεις:

| **Ερωτηματικό**                | **Αρνητικό**               | **Άγνωστο (γενικό)**             | **Άγνωστο (συγκεκριμένο)** | **Ολικός**              |
| ------------------------------ | -------------------------- | -------------------------------- | -------------------------- | ----------------------- |
| **ce** _(who/what)_            | **ane** (κανένας/τίποτα)   | **aye** (οποιοσδήποτε/οτιδήποτε) | **eye** (κάποιος/κάτι)     | **are** (όλοι/τα πάντα) |
| **cuál** _(which)_             | **anuál** (κανένα)         | **ayuál** (οποιοδήποτε)          | **eyuál** (κάποιο)         | **aruál** (όλα)         |
| **cove** _(where)_             | **anove** (πουθενά)        | **ayove** (οπουδήποτε)           | **eyove** (κάπου)          | **arove** (παντού)      |
| **cuando** _(when)_            | **anuando** (ποτέ)         | **ayuando** (οποιαδήποτε στιγμή) | **eyuando** (κάποτε)       | **aruando** (πάντα)     |
| **cuót** _(how much/how many)_ | **anuót** (κανένα)         | **ayuót** (οποιοδήποτε ποσό)     | **eyuót** (κάμποσο)        | **aruót** (όλα)         |
| **cume** _(how)_               | **anume** (κανένας τρόπος) | **ayume** (οποιοσδήποτε τρόπος)  | **eyume** (κάποιος τρόπος) | **arume** (κάθε τρόπος) |

- [x] **Σύστημα προθέσεων:**

    - **an-** → Αρνητική μορφή (απουσία ή άρνηση).
    - **ay-** → Αγνωστο γενικό (οποιοδήποτε).
    - **ey-** → Αγνωστο συγκεκριμένο (κάποιο).
    - **ar-** → Συλλογική αναφορά (όλα).
