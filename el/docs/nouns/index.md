# Ουσιαστικά

Τα ουσιαστικά αποτελούν τη **βάση** της γραμματικής της Αρκάδιας, ταξινομημένα με βάση τα **χαρακτηριστικά και τη χρήση τους**.

### **Ταξινόμηση με βάση την Ειδικότητα**

Η Αρκάδια διακρίνει τα ουσιαστικά με βάση το **βαθμό ειδικότητας** τους:

- **Κύρια ουσιαστικά** → Αναφέρονται σε μια **μοναδική οντότητα** (_Αρκάδια, Μαρία_).
- **Κοινά ουσιαστικά** → Περιγράφουν μια **γενική έννοια** (_δέντρο, μήλο_).

### **Γένος και Προαιρετικά Προθέματα**

Η Αρκάδια **δεν είναι μια έμφυλη γλωσσα**, αν και τα **προθέματα γένους** μπορούν να χρησιμοποιηθούν για σαφήνεια.
Αυτά τα προθέματα είναι **προαιρετικά** και χρησιμεύουν μόνο **για να προσδιορίσουν το γένος** όταν είναι απαραίτητα, όχι ως γραμματικές αναγκαιότητες.
Περισσότερες λεπτομέρειες σχετικά με τη **διαχείριση του γένους** είναι διαθέσιμες στον **[οδηγό γένους][gender]**.

### **Κατηγορίες Ουσιαστικών**

Η Αρκάδια ταξινομεί τα ουσιαστικά σε **πέντε γραμματικές [τάξεις][class]**, οι οποίες εξυπηρετούν ένα **δομικό ρόλο** χωρίς **σημασιολογική επίδραση**:

- **τάξη -i**
- **τάξη -e**
- **τάξη -a**
- **τάξη -o**
- **τάξη -u**

### **Γραμματικοί Αριθμοί**

Η Αρκάδια αναγνωρίζει **διακρίσεις μετρησιμότητας**, που επηρεάζουν τη συμπεριφορά των ουσιαστικών στην πληθυντική και στη μέτρηση:

- **Μετρήσιμα ουσιαστικά** → Μπορούν να **μετρηθούν** (_τρία μήλα, μισή ώρα_).
- **Μη μετρήσιμα ουσιαστικά** → Δεν μπορούν να **μετρηθούν μεμονωμένα** (_νερό, αγάπη_).

Περαιτέρω, υπάρχουν **πέντε γραμματικοί [αριθμοί][number]**.

### **Πτώσεις**

Η Αρκάδια χρησιμοποιεί **έντεκα γραμματικές πτώσεις**, καθορίζοντας το **ρόλο** κάθε ουσιαστικού μέσα σε μια πρόταση.
Λεπτομέρειες για τις λειτουργίες και τις αλληλεπιδράσεις τους μπορείτε να βρείτε **[εδώ][case]**.

### **Δημιουργία και Παράγωγα Ουσιαστικών**

Η Αρκάδια διαθέτει ένα **σύστημα δημιουργίας νέων ουσιαστικών** από υπάρχοντα, επιτρέποντας τη δομημένη επέκταση του λεξιλογίου.
Ο πλήρης κατάλογος των παραγωγικών κανόνων είναι διαθέσιμος στον **[οδηγό δημιουργίας ουσιαστικών][generation]**.
