# Πτώσεις

Η Αρκάδια έχει **επτά πτώσεις**, οι οποίες δηλώνονται με μια **κατάληξη**.
Αυτές οι πτώσεις καθορίζουν τον **συντακτικό ρόλο** ενός ουσιαστικού μέσα σε μια πρόταση.

Θα χρησιμοποιήσουμε το **"pan"** (_ψωμί_) ως παράδειγμα.

## **Πτώσεις**

- **Ονομαστική** (**"-is"**) → Δηλώνει το **υποκείμενο** της πρότασης: _panis_
- **Αιτιατική** (**"-on"**) → Δηλώνει το **άμεσο αντικείμενο**: _panon_
- **Δοτική** (**"-im"**) → Δηλώνει το **έμμεσο αντικείμενο** (παραλήπτης): _panim_
- **Αφαιρετική** (**"-a"**) → Δηλώνει το **έμμεσο αντικείμενο** (δωρητής/πηγή): _pana_
- **Κατηγορηματική** (**"-i"**) → Χρησιμοποιείται με **συνδετικά ρήματα** για να εκφράσει **το κατηγορούμενο**: _pani_
- **Προθετική** (**"-e"**) → Χρησιμοποιείται μετά από **προθέσεις**: _pane_
- **Κλητική** (**"-o"**) → Χρησιμοποιείται για **προσφώνηση**: _pano_

## **Κανόνες συγχώνευσης φωνηέντων**

Αν ένα ουσιαστικό **τελειώνει σε οπισθόφθογγο** (_a, o, u_), η κατάληξη συγχωνεύεται με το φωνήεν:

- [x] **Ονομαστική & Αιτιατική:** To **φωνήεν της κατάληξης** αφαιρείται (_αντί για "luna-is" → "lunas"_).
- [x] **Υπόλοιπες πτώσεις:** Το **τελευταίο φωνήεν του ουσιαστικού** αφαιρείται (_αντί για "luna-on" → "lunon"_).
