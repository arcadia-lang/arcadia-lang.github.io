# Αριθμός

Οι αριθμοί σχηματίζονται προσθέτοντας ένα **επίθημα** στη ρίζα της λέξης.
Η σημασιολογική έννοια του επιθήματος **διαφέρει** ανάλογα με το αν το ουσιαστικό είναι **μετρήσιμο ή μη μετρήσιμο**.

---

## **Μετρήσιμα ουσιαστικά**

Τα μετρήσιμα ουσιαστικά είναι αυτά που μπορούν να **ποσοτικοποιηθούν ως διακριτές μονάδες**, όπως μήλα, κιλά και άνθρωποι.
Χρησιμοποιώντας το **"libri"** (_βιβλίο_), τα επιθήματα είναι:

| Αριθμός     | Επίθημα | Παράδειγμα | Σημασία                                          |
| ----------- | ------- | ---------- | ------------------------------------------------ |
| Ενικός      | —Ø      | libri      | «ακριβώς ένα βιβλίο»                             |
| Κλασματικός | -d      | librid     | «μέρος ενός βιβλίου» (<1 μονάδα)                 |
| Πληθυντικός | -s      | libris     | «βιβλία» (>1, συμπεριλαμβανομένων των κλασμάτων) |
| Μηδενικός   | -n      | librin     | «κανένα βιβλίο»                                  |
| Ολικός      | -l      | libril     | «όλα τα βιβλία»                                  |

**Αρνητικές ποσότητες** παίρνουν το **ίδιο επίθημα** με την θετική τους αντίστοιχη.

---

## **Μη μετρήσιμα ουσιαστικά**

| Αριθμός     | Επίθημα | Παράδειγμα | Σημασία                           |
| ----------- | ------- | ---------- | --------------------------------- |
| Ενικός      | —Ø      | amore      | «αγάπη» (μη καθορισμένη ποσότητα) |
| Κλασματικός | -d      | amored     | «λίγη αγάπη»                      |
| Πληθυντικός | -s      | amores     | «άφθονη αγάπη»                    |
| Μηδενικός   | -n      | amoren     | «καθόλου αγάπη»                   |
| Ολικός      | -r      | amorer     | «όλη η αγάπη»                     |
