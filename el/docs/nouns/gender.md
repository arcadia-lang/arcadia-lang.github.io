# Γένος

Η Αρκαδία δεν είναι μια **έμφυλη γλώσσα** και δεν απαιτεί τα ουσιαστικά να δηλώνουν αν αναφέρονται σε **αρσενικό** ή **θηλυκό**.
Τα έμφυλα προθέματα είναι **προαιρετικά** και χρησιμοποιούνται **μόνο** όταν το φύλο είναι σχετικό με το περιεχόμενο.

## Έμφυλα Προθέματα

Τα παρακάτω **προαιρετικά** προθέματα μπορούν να προστεθούν στα ουσιαστικά για να **δηλώσουν το φύλο**:

- **pa-**: Δηλώνει αρσενικό φύλο (π.χ., _pafili_ = "γιός").
- **ma-**: Δηλώνει θηλυκό φύλο (π.χ., _mafili_ = "κόρη").

Όταν το φύλο δεν είναι **σχετικό**, τα ουσιαστικά παραμένουν **μη επισημασμένα** (π.χ., _fili_ = "παιδί").

!!! note "Χρήση Μη-Γραμματικού Φύλου"

    Τα έμφυλα προθέματα δεν θα πρέπει να χρησιμοποιούνται εκτός αν είναι **ρητά** απαραίτητα για **σαφήνεια** ή **έμφαση**.
    Για παράδειγμα, το **μη-έμφυλο** ουσιαστικό _fili_ είναι **προτιμότερο στις περισσότερες περιπτώσεις**, ενώ τα _pafili_ ή _mafili_ χρησιμοποιούνται μόνο όταν η προσδιορισμός του φύλου είναι σχετικός.
