# Τάξη

Η Αρκάδια κατηγοριοποιεί τα ουσιαστικά σε **πέντε γραμματικές τάξεις**, καθοριζόμενες από την **κατάληξη της λέξης**.
Ενώ η τάξη στην οποία ανήκει το ουσιαστικό **δεν αλλάζει το νόημα του**, η γλώσσα ακολουθεί μια **συνεπή και προβλέψιμη δομή**.

### **Οι Πέντε Τάξεις**

- **Στερεή (`-i`)** → Συνήθως συνδέεται με **δομημένες ή σταθερές οντότητες**.
- **Υγρή (`-e`)** → Συχνά συνδέεται με το **ρευστό και προσαρμοστικό**.
- **Αέρια (`-a`)** → Αναπαριστά κάτι **διασκορπισμένο ή εφήμερο**.
- **Πλάσμα (`-o`)** → Χρησιμοποιείται για **δυναμικές, έντονα ενεργητικές έννοιες**.
- **Εξωτική (`-u`)** → Καλύπτει **αφηρημένες ή ακαθόριστες οντότητες**.

Αυτές οι τάξεις εξυπηρετούν μια **καθαρά γραμματική λειτουργία**, αλλά το λεξιλόγιο της Αρκάδια τείνει να **ευθυγραμμίζεται φυσικά** με αυτές.
Για παράδειγμα, τα χερσαία ζώα είναι συνήθως **στερεά**, ενώ τα υδρόβια ζώα τείνουν να είναι **υγρά**—αντικατοπτρίζοντας τη σύνδεσή τους με το περιβάλλον.
