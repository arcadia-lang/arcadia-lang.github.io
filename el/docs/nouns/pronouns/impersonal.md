# Απρόσωπες Αντωνυμίες

Η Αρκαδία έχει **δύο απρόσωπες αντωνυμίες**, **"le"** και **"alé"**, οι οποίες διαφέρουν σε **βαθμό ειδικότητας**.

- [x] **"le"** → Αντιπροσωπεύει ένα **υπονοούμενο αλλά σχετικά συγκεκριμένο** αόριστο υποκείμενο (π.χ. μια γνωστή γενική ομάδα).
- [x] **"alé"** → Αντιπροσωπεύει ένα **εντελώς αόριστο** ή **διάχυτο** υποκείμενο (π.χ. η κοινωνία στο σύνολό της).

---

## **Κλίσεις Απρόσωπων Αντωνυμιών**

Τόσο το **"le"** όσο και το **"alé"** ακολουθούν τα πρότυπα κλίσης των ουσιαστικών.

### **_le_ (Συγκεκριμένο Απρόσωπο)**

| **Πτώση/Αριθμός**  | **Ενικός** | **Κλασματικός** | **Πληθυντικός** |
| ------------------ | ---------- | --------------- | --------------- |
| **Ονομαστική**     | leis       | ledis           | lesis           |
| **Αιτιατική**      | leon       | ledon           | leson           |
| **Δοτική**         | leim       | ledim           | lesim           |
| **Αφαιρετική**     | lea        | leda            | lesa            |
| **Κατηγορηματική** | lei        | ledi            | lesi            |
| **Προθετική**      | le         | lede            | lese            |
| **Κλητική**        | leo        | ledo            | leso            |

---

### **_alé_ (Αόριστο Απρόσωπο)**

| **Πτώση/Αριθμός**  | **Ενικός** | **Κλασματικός** | **Πληθυντικός** |
| ------------------ | ---------- | --------------- | --------------- |
| **Ονομαστική**     | aleis      | aledis          | alesis          |
| **Αιτιατική**      | aleon      | aledon          | aleson          |
| **Δοτική**         | aleim      | aledim          | alesim          |
| **Αφαιρετική**     | alea       | aleda           | alesa           |
| **Κατηγορηματική** | alei       | aledi           | alesi           |
| **Προθετική**      | alé        | alede           | alese           |
| **Κλητική**        | aleo       | aledo           | aleso           |
