# Αντωνυμίες

Η Αρκάδια έχει τέσσερις τύπους αντωνυμιών:

- **[Προσωπικές αντωνυμίες][personal]** – Χρησιμοποιούνται για να αναφερθούν σε άτομα στην προφορικό ή τον γραπτό λόγο.
- **[Απρόσωπες αντωνυμίες][impersonal]** – Μιλάνε για γενικά ή αόριστα πρόσωπα.
- **[Δεικτικές αντωνυμίες][demonstrative]** – Δείχνουν συγκεκριμένα αντικείμενα ή ιδέες σε σχέση με τον χώρο ή το θέμα.
- **[Ερωτηματικές αντωνυμίες][interrogative]** – Χρησιμοποιούνται για να κάνουν ερωτήσεις σχετικά με **την ταυτότητα, την ποσότητα, την επιλογή ή τα χαρακτηριστικά** ενός ουσιαστικού.
