# Προσωπικές Αντωνυμίες

Η Αρκάδια έχει **τρεις βασικές προσωπικές αντωνυμίες**, οι οποίες αντιστοιχούν **στο πρώτο, στο δεύτερο και στο τρίτο πρόσωπο**.
Το **τέταρτο πρόσωπο** χρησιμοποιείται για μια **γενικευμένη αναφορά**, εκφράζοντας **όλα τα αντικείμενα ή κανένα**—διαχειρίζεται μέσω **αόριστων αντωνυμιών**, παρά με μια ξεχωριστή προσωπική μορφή.

## **Επισκόπηση Αντωνυμιών**

| **Πρόσωπο** | **Αντωνυμία** |
| ----------- | ------------- |
| **1ο**      | me            |
| **2ο**      | ve            |
| **3ο**      | asé           |

## **Πλήρης Πίνακας Κλίσης**

Οι προσωπικές αντωνυμίες ακολουθούν τα **τυπικά πρότυπα κλίσης των ουσιαστικών**.

---

### **1ο Πρόσωπο ("me")**

| **Πτώση/Αριθμός**  | **Ενικός** | **Πληθυντικός** | **Πληθυντικός (συμπεριληπτικός)** |
| ------------------ | ---------- | --------------- | --------------------------------- |
| **Ονομαστική**     | meis       | mesis           | emesis                            |
| **Αιτιατική**      | meon       | meson           | emeson                            |
| **Δοτική**         | meim       | mesim           | emesim                            |
| **Αφαιρετική**     | mea        | mesa            | emesa                             |
| **Κατηγορηματική** | mei        | mesi            | emesi                             |
| **Προθετική**      | me         | mese            | emese                             |
| **Κλητική**        | meo        | meso            | emeso                             |

---
