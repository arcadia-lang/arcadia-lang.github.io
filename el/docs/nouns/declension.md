# Πλήρης κλίση

Η κλίση των ουσιαστικών στην Αρκάδια είναι πολύ κανονική, επομένως οι πίνακες κλίσης δεν είναι πολύ χρήσιμοι.
Παρέχονται εδώ για πληρότητα.

## τάξη -i

| Πτώση/Αριθμός  | Ενικός   | Κλασματικός | Πληθυντικός | Μηδενικός | Ολικός    |
| -------------- | -------- | ----------- | ----------- | --------- | --------- |
| Ονομαστική     | libris   | libridis    | librisis    | librinis  | librilis  |
| Εργατική       | librier  | librider    | libriser    | libriner  | libriler  |
| Απόλυτη        | librim   | libridim    | librisim    | librinim  | librilim  |
| Αιτιατική      | librion  | libridon    | librison    | librinon  | librilon  |
| Δοτική         | libril   | libridil    | librisil    | librinil  | librilil  |
| Θεματική       | libriath | libridath   | librisath   | librinath | librilath |
| Κατηγορηματική | libri    | libridi     | librisi     | librini   | librili   |
| Προθετική      | librie   | libride     | librise     | librine   | librile   |
| Κλητική        | librio   | librido     | libriso     | librino   | librilo   |
| Οργανική       | libria   | librida     | librisa     | librina   | librila   |
| Ωφελητική      | libriu   | libridu     | librisu     | librinu   | librilu   |
