# Παράγωγα

Η κατανόηση των **ετυμολογικών αρχών** της Αρκάδιας είναι απαραίτητη για την αποτελεσματική παραγωγή νέων λέξεων.

Δεδομένου ότι το λεξιλόγιο της Αρκάδιας είναι **μικρότερο από τις περισσότερες γλώσσες**, η παραγωγή λέξεων συμβαίνει πιο συχνά.
Το σύστημα έχει σχεδιαστεί για να **ελαχιστοποιεί τα λεξικά κενά**, διασφαλίζοντας ότι οι απαραίτητες λέξεις μπορούν πάντα να σχηματιστούν.

Οι περισσότερες λέξεις προέρχονται από **ρήματα**, καθιστώντας την **παράγωγη από ρήματα** μια βασική διαδικασία.
Αυτή η ενότητα είναι χωρισμένη με βάση τον **τύπο λέξης που προκύπτει**:

- **[Ρήματα][verbs]**
- **[Ουσιαστικά][nouns]**

[verbs]: ./verbs.md
[nouns]: ./nouns.md
