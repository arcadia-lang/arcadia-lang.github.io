## Σημεία στίξης

| Σημεία στίξης | Όνομα            | Χρήση                                                                           |
| ------------- | ---------------- | ------------------------------------------------------------------------------- |
| .             | Τελεία           | Tελειώνει μια οριστική πρόταση.                                                 |
| ;             | Άνω τελεία       | Συνδέει σχετικές ανεξάρτητες προτάσεις ή χωρίζει στοιχεία σε μια σύνθετη λίστα. |
| ,             | Κόμμα            | Χωρίζει τα στοιχεία σε μια λίστα ή τις προτάσεις μέσα σε μια πρόταση.           |
| :             | Διπλή Τελεία     | Εισάγει μια λίστα, παράθεση ή εξήγηση.                                          |
| !             | Θαυμαστικό       | Εκφράζει έντονα συναισθήματα ή μια εντολή.                                      |
| ?             | Ερωτηματικό      | Δηλώνει μια ερώτηση.                                                            |
| ""            | Διπλά Εισαγωγικά | Περικλείουν άμεσες ομιλίες, παραθέσεις ή τίτλους.                               |
