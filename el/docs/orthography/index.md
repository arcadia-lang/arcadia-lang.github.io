# Ορθογραφία της Αρκάδια

Η Αρκάδια χρησιμοποιεί **23 γράμματα** από το λατινικό αλφάβητο, **εξαιρώντας τα k, q και w**.
Κάθε γράμμα έχει μια **σταθερή προφορά**, εξασφαλίζοντας μια **συνεπή αντιστοιχία ορθογραφίας προς ήχο**.

## Δίψηφα

Η Αρκάδια χρησιμοποιεί επίσης τα παρακάτω **δίψηφα**:

| Δίψηφο | ΔΦΑ  | Ελληνικό Παρόμοιο |
| ------ | ---- | ----------------- |
| ch     | /t͜s/ | **τσ**άι          |
| dh     | /ð/  | **δ**έντρο        |
| th     | /θ/  | **θ**άλασσα       |

!!! tip "Κανόνας"

    Το φώνημα **/ks/** γράφεται πάντα ως **"x"**, ποτέ ως "cs."

!!! warning "Προφορά"

    Κάθε γράμμα προφέρεται σύμφωνα με την **ΔΦΑ** του και μπορεί να διαφέρει ελαφρώς από την ελληνική προφορά.
    Το **ελληνικό ισοδύναμο** παρέχει μόνο την πιο κοντινή προσέγγιση.

## Τονισμός

Η Αρκάδια χρησιμοποιεί την **οξεία (´)** για να δηλώσει τονισμό:

- Λέξεις **χωρίς οξεία** → Ο τόνος είναι στη **παραλήγουσα** (ή στη **μοναδική συλλαβή** αν η λέξη είναι μονοσύλλαβη).
- Λέξεις **με οξεία** → Ο τόνος είναι στη **σημειωμένη συλλαβή**.

Παραδείγματα:

- **homis** (_άνθρωπος_) → τόνος στο "o" (**ho**mis).
- **ni** (_όχι_) → τόνος στο "i" (**ni**).
- **asé** (_προσωπική αντωνυμία τρίτου προσώπου_) → τόνος στο "é" (as**é**).

## Συγχώνευση Φωνηέντων & Μακρό

Όταν **συνδυάζονται λέξεις** ή **προστίθενται επιθήματα**, **παρόμοια γειτονικά φωνήεντα** **δεν επιτρέπονται ποτέ**—πάντα **συγχωνεύονται σε μια μόνο περίπτωση**.

!!! tip "Κανόνας"

    Στις **λεξικογραφικές καταχωρήσεις, τα εγχειρίδια γραμματικής και τους οδηγούς σύνταξης**, οι συγχωνεύσεις φωνηέντων μπορούν να σημειωθούν με ένα **μακρό** πάνω από το φωνήεν (π.χ. _mē_ αντί για _me_).
    Ωστόσο, **τα μακρό δεν χρησιμοποιούνται στη φυσιολογική γραφή**—οι λέξεις γράφονται χωρίς αυτά στο καθημερινό κείμενο.

## Σημεία στίξης & Μορφοποίηση

Η Αρκάδια έχει **κανένα κεφαλαίο γράμμα** και **καμία σιωπηλή λέξη**.
Χρησιμοποιεί τα παρακάτω **σημεία στίξης**:

| Σημείο Στίξης | Όνομα            | Χρήση                                                                           |
| ------------- | ---------------- | ------------------------------------------------------------------------------- |
| .             | Τελεία           | Tελειώνει μια οριστική πρόταση.                                                 |
| ;             | Άνω τελεία       | Συνδέει σχετικές ανεξάρτητες προτάσεις ή χωρίζει στοιχεία σε μια σύνθετη λίστα. |
| ,             | Κόμμα            | Χωρίζει τα στοιχεία σε μια λίστα ή τις προτάσεις μέσα σε μια πρόταση.           |
| :             | Διπλή Τελεία     | Εισάγει μια λίστα, παράθεση ή εξήγηση.                                          |
| !             | Θαυμαστικό       | Εκφράζει έντονα συναισθήματα ή μια εντολή.                                      |
| ?             | Ερωτηματικό      | Δηλώνει μια ερώτηση.                                                            |
| ""            | Διπλά Εισαγωγικά | Περικλείουν άμεσες ομιλίες, παραθέσεις ή τίτλους.                               |

### **Παύλες & Απόστροφοι**

- **Παύλες (-)** → Χρησιμοποιείται για τις σύνθετες λέξεις που σχηματίζουν ένα μόνο λεξικό στοιχείο.
- **Απόστροφοι (’)** → Χρησιμοποιείται για την έκθλιψη και την αφαίρεση φωνηέντων.
- **Μονά Εισαγωγικά (')** → Μπορεί να αντικαταστήσει τους αποστροφούς για ευκολότερη πληκτρολόγηση.

!!! note "Σημεία στίξης & Διπλά Εισαγωγικά"

    Punctuation should be placed **outside** quotation marks **unless** both the quoted sentence and the main sentence **end simultaneously**.

## Κύρια Ονόματα & Δάνεια

- Τα κύρια ονόματα **διατηρούν την αρχική τους ορθογραφία** (αν είναι στο λατινικό αλφάβητο).
- Τα ονόματα από άλλα αλφάβητα **χρησιμοποιούν τη λατινική μεταγραφή** ή μια φωνητική προσαρμογή.
- Ακόμα και με την αρχική ορθογραφία, τα ονόματα προφέρονται **όσο το δυνατόν πιο κοντά** στη γλώσσα προέλευσης.
- Ως **βοήθεια του αναγνώστη**, ένα όνομα μπορεί να **μεταγραφεί μέσα σε διπλά εισαγωγικά** την πρώτη φορά που εμφανίζεται:
    - Παράδειγμα: "`mike::maic::`"
- Τα δάνεια μπορεί να περιέχουν τα γράμματα k, q ή w, αν και δεν εμφανίζονται σε αρκαδικές λέξεις.
