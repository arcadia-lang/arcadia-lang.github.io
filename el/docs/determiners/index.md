# Προσδιοριστές

Οι προσδιοριστές είναι λέξεις που προηγούνται ενός ουσιαστικού, παρέχοντας πρόσθετες πληροφορίες σχετικά με **την αναφορά, την ποσότητα ή την ειδικότητα** του.

Η Αρκάδια έχει **έξι τύπους** προσδιοριστών:

- **[Άρθρα][article]** → Ορίζει την ειδικότητα: _το_ δέντρο, _ένα_ μήλο.
- **[Δεικτικοί προσδιοριστές][demonstrative determiner]** → Δηλώνει εγγύτητα: _αυτή_ η μηχανή, _εκείνος_ ο άνθρωπος.
- **[Ερωτηματικοί προσδιοριστές][interrogative determiner]** → Δημιουργεί ερωτήσεις: _ποιο_ βιβλίο, _ποιο_ πάπλωμα.
- **[Κτητικά][possessive]** → Εκφράζει ιδιοκτησία: _Η_ τσάντα _της Μαρίας_, _το_ κρεβάτι _μου_.
- **[Ποσοδείκτες][quantifier]** → Περιγράφει ποσότητα: _μερικοί_ σκύλοι, _κάθε_ παιδί.
- **[Αριθμητικά][numeral]** → Καθορίζει έναν ακριβή αριθμό: _δύο_ δελφίνια, _ένα εκατομμύριο_ δολάρια.

- [x] **Οι προσδιοριστές μπορούν να συνδυαστούν**, όπως στο **"οι τρεις σωματοφύλακες."**

### **Κανόνες προσδιοριστών στην Αρκάδια**

- [x] **Ένας προσδιοριστής είναι υποχρεωτικός** εκτός αν το ουσιαστικό είναι σε **μηδενικό**, ή **συλλογικό αριθμό**, όπου **δεν χρησιμοποιείται συνήθως προσδιοριστής**.
- [x] **Τα κύρια ονόματα** δεν απαιτούν προσδιοριστή αλλά **μπορούν να πάρουν έναν** όταν χρειάζεται να τονιστεί η ειδικότητα.
