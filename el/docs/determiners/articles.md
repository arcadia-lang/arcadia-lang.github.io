# Άρθρα

Η Αρκαδία έχει **πέντε άρθρα**, το καθένα με διαφορετική γραμματική λειτουργία:

- **Οριστικό (`la`)** → _la leon_ → **"Το λιοντάρι"** (ένα συγκεκριμένο, γνωστό λιοντάρι).
- **Οριστικό-Αναφορικό (`le`)** → _le leon_ → **"Το λιοντάρι (για το οποίο μιλήσαμε νωρίτερα)"** (αναφορά σε ένα λιοντάρι που έχει αναφερθεί προηγουμένως).
- **Αόριστο-Αδύνατο (`ya`)** → _ya leon_ → **"Οποιοδήποτε λιοντάρι"** (γενική αναφορά σε οποιοδήποτε λιοντάρι).
- **Αόριστο-Ισχυρό (`ye`)** → _ye leon_ → **"Ένα λιοντάρι (ένα συγκεκριμένο αλλά άγνωστο)"** (ένα συγκεκριμένο αλλά συγκεκριμένο λιοντάρι).
- **Ολικός (`ta`)** → _ta leon_ → **"Μια λεοπαρδαλή λιονταριών"** (συγκεντρωτική συλλογική έννοια).

### **Κανόνες Χρήσης**

✔ **Το οριστικό άρθρο (`la/le`)** μπορεί να τροποποιήσει **αόριστους προσδιοριστές**, μετατρέποντάς τους σε **οριστικούς**:

- _tre homosis_ → **"τρεις άνθρωποι"**
- _las tre homosis_ → **"τρεις συγκεκριμένοι άνθρωποι"**

✔ **Το αόριστο άρθρο (`ya/ye`)** μπορεί να μετατρέψει τα ονόματα σε **κοινά ονόματα**:

- _ye adamis_ → **"κάποιος που λέγεται Αδάμ"**

---

## **Παραδείγματα**

1. _convenivo adamon._

    - **Μετάφραση:** "Γνώρισα τον Αδάμ."
    - **Εξήγηση:** Αναφέρεται σε **συγκεκριμένο άτομο που ονομάζεται Αδάμ** και το οποίο γνωρίζει ο ομιλητής.

1. _convenivo le adamon._

    - **Μετάφραση:** "Γνώρισα τον Αδάμ."
    - **Εξήγηση:** Αναφέρεται σε **συγκεκριμένο άτομο που ονομάζεται Αδάμ** και το οποίο έχει αναφερθεί προηγουμένως.

1. _convenivo ye adamon._

    - **Translation:** "Γνώρισα κάποιον που λέγεται Αδάμ."
    - **Explanation:** Αναφέρεται σε **κάποιον που ονομάζεται Αδάμ**, χωρίς να υποθέτει οικειότητα.

### **Προσαρμογές Προφοράς**

✔ Εάν ένα ενικό άρθρο **προηγείται ενός ουσιαστικού που αρχίζει με το ίδιο φωνήεν**, **συμβάλλεται**, αντικαθιστώντας το φωνήεν με απόστροφο:

- **la** → **l'** (_la amigos_ → _l' amicus_)
- **le** → **l'** (_le energias_ → _l' energias_)
- **ya** → **y'** (_ya avis_ → _y' avis_)
- **ye** → **y'** (_ye enginas_ → _y' enginas_)
- **ta** → **t'** (_ta antecenus_ → _t' antecenus_)

Αυτό εξασφαλίζει **ομαλή προφορά** διατηρώντας παράλληλα τη σαφήνεια στην ομιλία.
