# Δεικτικά

Η Αρκαδία έχει τρία **δεσμευτικά στοιχεία δείξης**, που **ποτέ δεν χρησιμοποιούνται μόνα τους**.
Αντίθετα, **συνδέονται με ερωτηματικές λέξεις** για να προσθέσουν **χωρική ή χρονική δείξη**, προσδιορίζοντας την εγγύτητα ή την απόσταση.

Τα **τρία επίπεδα δείξης** είναι:

- **_est:_** Κοντά (κοντά στον ομιλητή).
- **_ess:_** Μακριά (ορατό αλλά μακρινό).
- **_tis:_** Πολύ μακριά ή αόρατο (εκτός οπτικού πεδίου, αφηρημένο ή μακρινό στο χρόνο).

Δεδομένου ότι **est, ess και tis** είναι εξαρτημένες τροποποιήσεις, **δεν μπορούν να εμφανιστούν ανεξάρτητα** στην ομιλία ή τη γραφή.
Πάντα απαιτούν μια **ερωτηματική βάση** για να λειτουργήσουν σωστά.

### **Ερωτηματικά Δεικτικά**

Τα στοιχεία δείξης συνδυάζονται με **ερωτηματικές λέξεις** για να σχηματίσουν ακριβή δεικτικά:

- **_ce:_** (ποιος/τι)

    - **Κοντά:** _este_ → "Αυτό"
    - **Μακριά:** _esse_ → "Εκείνο"
    - **Πολύ μακριά/αόρατο:** _tise_ → "Αυτό που είναι εκτός του οπτικού μου πεδίου"

- **_cuál:_** (ποιο απ' αυτά)

    - **Κοντά:** _estuál_ → "Αυτό"
    - **Μακριά:** _essuál_ → "Εκείνο"
    - **Πολύ μακριά/αόρατο:** _tisuál_ → "Αυτό που είναι εκτός του οπτικού μου πεδίου"

- **_cove:_** (πού)

    - **Κοντά:** _estove_ → "Εδώ"
    - **Μακριά:** _essove_ → "Εκεί"
    - **Πολύ μακριά/αόρατο:** _tisove_ → "Το μέρος που είναι εκτός οπτικού πεδίου"

- **_cuando:_** (πότε)

    - **Κοντά:** _estuando_ → "Τώρα"
    - **Μακριά:** _essuando_ → "Τότε"
    - **Πολύ μακριά/αόρατο:** _tisuando_ → "Σε μακρινό χρόνο"

- **_cuót:_** (πόσα)

    - **Κοντά:** _estuót_ → "Τόσο"
    - **Μακριά:** _essuót_ → "Τόσο"
    - **Πολύ μακριά/αόρατο:** _tisuót_ → "Τόσο"

- **_cume:_** (πώς)

    - **Κοντά:** _estume_ → "Έτσι"
    - **Μακριά:** _essume_ → "Αλλιώς"
    - **Πολύ μακριά/αόρατο:** _tisume_ → "Αλλιώς"

!!! note "Δεικτικές Εκφράσεις"

    Τα _"-ove"_, _"-uando"_ και _"-ume"_ **δεν είναι αληθινά δεικτικά**, αλλά περιλαμβάνονται εδώ λόγω της στενής τους σχέσης με τις **δεικτικές εκφράσεις**.

---

## **Κανόνες Συμφωνίας**

- [x] Όταν συνδυάζεται με ένα ουσιαστικό, **το δεικτικό πρέπει να συμφωνεί σε αριθμό**.
- [x] Τα δεικτικά **δεν κλίνονται** όταν χρησιμοποιούνται με ένα ουσιαστικό (για να αποφευχθεί η πλεονασμός).
- [x] Όταν **χρησιμοποιούνται ανεξάρτητα**, η κλίση **είναι υποχρεωτική**.

---

## **"ess" vs. "tis" σε Χρονικά Πλαίσια**

- [x] **"essando"** → Χρησιμοποιείται όταν αναφερόμαστε σε **ένα χρονικό διάστημα εντός μιας ζωής**.
- [x] **"tisando"** → Χρησιμοποιείται όταν αναφερόμαστε σε **ένα πιο μακρινό χρονικό διάστημα ή μια αφηρημένη έννοια**.

---

## **Παραδείγματα**

- [x] **Δεικτικό με ουσιαστικό** → Καμία κλίση δεν απαιτείται:

    - _este pomos_ → **"Αυτό το μήλο"**
    - _estes pomosis_ → **"Αυτά τα μήλα"**
    - _estuál pomos_ → **"Αυτό εδώ το μήλο"** (επιλέγοντας ανάμεσα σε μήλα)
    - _estót aguas_ → **"Τόσο νερό"**

- [x] **Δεικτικό ως αυτόνομη αντωνυμία** → Απαιτεί κλίση:

    - _volavo esseon._ → **"Θέλω αυτό."** (Απαιτείται κλίση)
    - _volavo tiseon._ → **"Θέλω αυτό που είναι εκτός του οπτικού μου πεδίου."**
