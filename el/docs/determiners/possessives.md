# Κτητικά

Η Αρκάδια χρησιμοποιεί τις προθέσεις **"de"** και **"du"** για να δηλώσει την κατοχή.

- [x] **Τα κτητικά λειτουργούν ως προσδιοριστές** και εμφανίζονται **πριν από το ουσιαστικό**.
- [x] **Ο κατοχός πρώτα** → **Ο κάτοχος προηγείται πάντα του κατεχόμενου** στη σύνταξη φράσεων.

### **Αλλοτριωτή και Μη Αλλοτριωτή Κατοχή**

- [x] **"de"** → Δηλώνει **μη αλλοτριωτή κατοχή**, όπου ο κάτοχος είναι **αναπόσπαστο μέρος** του κατεχόμενου.
- [x] **"du"** → Δηλώνει **αλλοτριωτή κατοχή**, όπου ο κάτοχος **δεν είναι εγγενώς συνδεδεμένος** με το κατεχόμενο.

- [x] **Δεν υπάρχουν κτητικές αντωνυμίες** → Αντίθετα, η κατοχή **εκφράζεται με "de" ή "du" + αντωνυμία**.

---

## **Παραδείγματα**

- [x] **Αλλοτριωτή Κατοχή (`du`)**

    - _save du me libris magne_ → **"Το βιβλίο μου είναι μεγάλο."**
    - **Εξήγηση:** Το βιβλίο **δεν είναι εγγενώς μέρος** του ομιλητή.

- [x] **Μη Αλλοτριωτή Κατοχή (`de`)**

    - _seve de me de file viparenis adamo_ → **"Ο πατέρας του παιδιού μου είναι ο Αδάμ."**
    - **Εξήγηση:** Η σχέση του παιδιού με τον πατέρα **είναι εγγενής**.

- [x] **Αλυσίδα Κτητικών**
    - _save ta libris du me_ → **"Το βιβλίο είναι δικό μου."**
    - **Εξήγηση:** Η αλυσίδα κατοχής ακολουθεί **τον κάτοχο πρώτα**.
