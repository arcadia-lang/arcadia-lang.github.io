# Ερωτήσεις

Οι ερωτήσεις στην Αρκάδια σχηματίζονται χρησιμοποιώντας ένα **ερωτηματικό μόριο**, ακολουθούμενο από μια καταφατική πρόταση **με την άγνωστη λέξη παραλειπόμενη**.

- [x] **Ναι/Όχι ερωτήσεις** →Υποδεικνύονται από **αυξανόμενη τονικότητα** στην προφορική Αρκάδια.
- [x] **Ανοιχτές ερωτήσεις** → Χρησιμοποιoύνται με **ερωτηματικά μόρια**, τα οποία συνδυάζονται με **προθέσεις και κλίσεις** με τον αναμενόμενο τρόπο.

## **Ερωτηματικά μόρια**

| **Μόριο**  | **Ελληνικά**  | **Σημασία**      |
| ---------- | ------------- | ---------------- |
| **cu**     |               | Ναι/Όχι ερώτηση  |
| **ce**     | Ποιος/τι      | Γενική ερώτηση   |
| **cuál**   | Ποιο από αυτά | Επιλογή          |
| **cove**   | Πού           | Χωροταξική       |
| **cuando** | Πότε          | Χρονική αναφορά  |
| **cuót**   | Πόσο          | Ποσοτική ερώτηση |
| **cume**   | Πώς           | Ερώτηση τρόπου   |

- [x] **Οι ερωτηματικές λέξεις μπορούν να συνδυαστούν με προθέσεις**, επιτρέποντας **διευρυμένες σημασίες**:

    - _pro ce_ → "Γιατί"
    - _por ce_ → "Για ποιου το όφελος"

!!! note "Διαχείριση Αγνώστων"

    Στην Αρκάδια, η υποβολή ερωτήματος **αφαιρεί τις επηρεαζόμενες καταλήξεις**:

        - Το **"cuót"** αφαιρεί **σημάδια ποσότητας**, απλοποιώντας τα ουσιαστικά.
        - Το **"cuando"** αφαιρεί **χρονικές καταλήξεις** όταν ρωτάμε για διάρκεια.
        - Το **"ce"** αφαιρεί **καταλήξεις προσώπου** όταν ρωτάμε για το υποκείμενο ενός ρήματος.

## **Παραδείγματα**

- [x] **Ναι/Όχι Ερώτηση:**

    - _cu donavi ta libron meim?_ → **"Μου δίνεις το βιβλίο;"**

- [x] **Ερώτηση μεθόδου:**

    - _cume donavi ta libron meim?_ → **"Πώς μου δίνεις το βιβλίο;"**
        - **Εξήγηση:** Ρωτώντας για τη μέθοδο—π.χ., με το χέρι, ταχυδρομικώς.

- [x] **Ερώτηση τοποθεσίας:**

- _cove donavi ta libron meim?_ → **"Πού μου δίνεις το βιβλίο;"**

- [x] **Ερώτηση υποκειμένου:**

- _ceim donavi ta libron?_ → **"Σε ποιον δίνεις το βιβλίο;"**
- _ceon donavi meim?_ → **"Τι μου δίνεις;"**

- [x] **Επιλογή:**

- _cualon libron donavi meim?_ → **"Ποιο βιβλίο μου δίνεις;"**

- [x] **Χρονική αναφορά:**

- _cuando donovi ta libron meim?_ → **"Πότε θα μου δώσεις το βιβλίο;"**
- **Εξήγηση:** Υπονοεί ότι είναι **στο μέλλον**.

- [x] **Ποσοτική Ερώτηση:**

- _cuót libron donavi meim?_ → **"Πόσα βιβλία μου δίνεις;"**

---

### **Ρωτώντας για το Υποκείμενο του Ρήματος**

- [x] **Διατηρεί μόνο τα γνωστά μέρη**, διασφαλίζοντας σαφήνεια στην αναφορά.

| **Πρόταση**                     | **Μετάφραση**                         | **Εξήγηση**                                                                       |
| ------------------------------- | ------------------------------------- | --------------------------------------------------------------------------------- |
| _ceis donav ta libron meim?_    | "Ποιος μου δίνει το βιβλίο;"          | Καμία ένδειξη για **ποιος ή πόσοι** εμπλέκονται.                                  |
| _cesis donaves ta libron meim?_ | "Ποιος μου δίνει το βιβλίο;"          | **Τρίτο πρόσωπο πληθυντικού**—πολλοί άνθρωποι, **εξαιρώντας εσένα ή εμένα**.      |
| _cesis donavis ta libron meim?_ | "Ποιος από εσάς μου δίνει το βιβλίο;" | Χρησιμοποιείται ως **ερώτηση παρακολούθησης** όταν το υποκείμενο δεν είναι σαφές. |

## **Συνηθισμένες Ερωτήσεις**

- [x] _cei seve de ve namis?_ → **"Πώς σε λένε;"**
- [x] _cume savi?_ → **"Πώς είσαι;"**
- [x] _ex cove sevi?_ → **"Από πού είσαι;"**
