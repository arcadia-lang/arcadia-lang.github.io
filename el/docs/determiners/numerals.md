# Αριθμητικά

Η Αρκαδία χρησιμοποιεί **δύο παράλληλα αριθμητικά συστήματα**:

- [x] **Δεκαδικό σύστημα** → Χρησιμοποιείται συνήθως για γενική μέτρηση και υπολογισμούς.
- [x] **Δεκαεξαδικό σύστημα** → Συμπεριλαμβάνεται για **μελλοντική προετοιμασία**, επιτρέποντας την επεκταμένη αριθμητική αναπαράσταση.

### **Αριθμοί και Γραφή**

- [x] Η **δεκαεξαδική σημειογραφία χρησιμοποιεί `0-9` και `A-F`**, διασφαλίζοντας ότι **δεν υπάρχει επικάλυψη με τις γραπτές λέξεις**, καθώς η Αρκαδία χρησιμοποιεί μόνο **μικρά γράμματα**.
- [x] Οι **αριθμοί ομαδοποιούνται χρησιμοποιώντας κάτω παύλες (`_`)** για σαφήνεια και **μη αμφίβολη αναγνωσιμότητα**.

## **Μεγαλύτεροι Αριθμοί**

- [x] Μετά το **999_999**, και τα δύο συστήματα χρησιμοποιούν **δυνάμεις του 1_000_000** για να δημιουργήσουν **ονόματα για αυθαίρετα μεγάλα νούμερα**.
- [x] Για **μορφές δυνάμεων με μία λέξη**, η Αρκαδία **αντικαθιστά το "ad X potente"** με το **επίθημα "-ente"**.

### **Δεκαδικά Ονόματα Δυνάμεων**

| **Δεκαδικό**      | **Όνομα**                                    |
| ----------------- | -------------------------------------------- |
| 1_000_000         | **milonu**                                   |
| 1_000_000_000_000 | **dosente milonu** _(milonu ad dos potente)_ |

### **Δεκαεξαδικά Ονόματα Δυνάμεων**

| **Δεκαεξαδικό**     | **Όνομα**                              |
| ------------------- | -------------------------------------- |
| 0x1_000_000         | **bet**                                |
| 0x1_000_000_000_000 | **dosente bet** _(bet ad dos potente)_ |
