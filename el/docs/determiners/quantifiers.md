# Ποσοδείκτες

Οι ποσοδείκτες είναι **προσδιοριστές** που υποδεικνύουν την **ποσότητα ενός ουσιαστικού**.

- [x] **Μηδενικός και συλλογικός αριθμός** → Φυσικά καθιστούν **"όλους" και "κανέναν" περιττούς**, απλοποιώντας τις εκφράσεις.
- [x] **Περιορισμοί Χρήσης** → **Οι ποσοδείκτες δεν μπορούν να χρησιμοποιηθούν με αμετρήσιμα ουσιαστικά**, καθώς **ο αριθμός ήδη μεταφέρει ποσότητα**.

### **Βασικοί Ποσοδείκτες**

Η Αρκάδια ορίζει τους παρακάτω **ποσοδείκτες**:

- **_poco_** → **"Λίγοι"** (μικρή αλλά διακριτή ποσότητα).
- **_aliame_** → **"Μερικοί"** (μπορεί να αναφέρεται σε ένα ή περισσότερα).
- **_oliame_** → **"Μερικοί"** (δεν μπορεί να αναφέρεται σε ένα, μόνο σε πληθυντικό).
- **_multe_** → **"Πολλοί"** (μεγάλη ποσότητα).
- **_cisca_** → **"Κάθε"** (ατομική έμφαση).

### **Σημειώσεις Χρήσης**

- [x] **"aliame"** → Αν και **μπορεί να αναφέρεται σε ένα**, πάντα **παίρνει μια πληθυντική μορφή ουσιαστικού**.
- [x] **"cisca"** →
    - Χρησιμοποιείται με **ενιαία ουσιαστικά** για να τονίσει **ατομικές οντότητες**.
    - Χρησιμοποιείται με **συλλογικά ουσιαστικά** για να τονίσει **τόσο τα άτομα όσο και την ομάδα**.

---

## **Παραδείγματα**

- [x] **Συλλογική Εστίαση (`puereris` σε συλλογική μορφή)**

    - _edravur puereris e pomon_ → **"Κάθε παιδί τρώει ένα μήλο."**
    - **Εξήγηση:** Το **συλλογικό ουσιαστικό** και το ρήμα τονίζουν **την ομάδα στο σύνολό της**.

- [x] **Ατομική Έμφαση (`cisca` με ενιαίο ουσιαστικό)**

    - _edrave cisca pueris e pomon_ → **"Κάθε παιδί τρώει ένα μήλο."**
    - **Εξήγηση:** Το ρήμα και το ουσιαστικό παραμένουν **ενιαία**, εστιάζοντας σε **ατομικά παιδιά**.

- [x] **Hybrid Emphasis (`cisca` with collective noun)**

    - _edravur cisca puereris e pomon_ → **"Κάθε παιδί τρώει ένα μήλο."**
    - **Εξήγηση:** Το **συλλογικό ουσιαστικό + cisca** τονίζει **τόσο τα άτομα όσο και την ομάδα**.
