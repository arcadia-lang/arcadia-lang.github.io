# Ρήματα

Τα ρήματα της Αρκάδια **καθορίζουν τη δομή της πρότασης**, διαμορφώνοντας τη **κύρια σύνταξη** με βάση τον **τύπο του ρήματος**.
Μεταφέρουν εκτενή γραμματική πληροφορία μέσω ενός **συστήματος προθέματος και επιθήματος**, ενσωματώνοντας **διακρίσεις διάθεσης, χρόνου, όψης και προσώπου**.

### **Βασικά Χαρακτηριστικά Ρημάτων**

- **[Τύποι ρημάτων][type]** → Ορίζουν τη δομή της πρότασης και καθορίζουν τις **συντακτικές εξαρτήσεις**.
- **[Εγκλίσεις][mood]** → Η Αρκαδία έχει **8 εγκλίσεις**, επιτρέποντας την ακριβή έκφραση της **πρόθεσης και της τροπικότητας**.
- **[Χρόνοι][tense]** → Η Αρκαδία διακρίνει **5 χρόνους**, δομώντας τις ενέργειες σε **χρονικά πλαίσια**.
- **[Όψεις][aspect]** → Με **9 όψεις**, τα ρήματα κωδικοποιούν τη **ροή και τη διάρκεια της ενέργειας**.
- **[Πρόσωπα][person]** → Λόγω των **μηδενικών και συλλογικών αριθμών**, η Αρκαδία έχει **4 γραμματικά πρόσωπα**.
