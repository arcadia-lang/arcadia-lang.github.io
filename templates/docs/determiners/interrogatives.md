# Questions

Questions in Arcadia are formed using a **question particle**, followed by an affirmative sentence **with the unknown part omitted**.

- [x] **Yes/no questions** → Indicated by **rising intonation** in spoken Arcadia.
- [x] **Open-ended questions** → Use **interrogative particles**, which combine with **prepositions and declensions** as expected.

## **Question Particles**

| **Particle** | **English**       | **Meaning**      |
| ------------ | ----------------- | ---------------- |
| **cu**       |                   | Yes/no question  |
| **ce**       | Who/what          | Generic inquiry  |
| **cuál**     | Which             | Choice selection |
| **cove**     | Where             | Location-based   |
| **cuando**   | When              | Time reference   |
| **cuót**     | How much/how many | Quantity inquiry |
| **cume**     | How               | Manner inquiry   |

- [x] **Interrogative words can combine with prepositions**, allowing **expanded meanings**:
  - _pro ce_ → "Why"
  - _por ce_ → "For whose benefit"

!!! note "Handling Unknowns"

    In Arcadia, asking a question **causes affected suffixes to drop**:

        - **"&{cuót}"** removes **quantity markers**, simplifying nouns.
        - **"&{cuando}"** drops **time suffixes** when asking about duration.
        - **"&{ce}"** removes **person suffixes** when querying the subject of a verb.

---

## **Examples**

- [x] **Yes/No Question:**
  - _cu donavi ta libron meim?_ → **"Are you giving the book to me?"**

- [x] **Method Inquiry:**
  - _cume donavi ta libron meim?_ → **"How are you giving the book to me?"**
    - **Explanation:** Asking about the method—e.g., by hand, by post.

- [x] **Location Inquiry:**

- _cove donavi ta libron meim?_ → **"Where are you giving the book to me?"**

- [x] **Subject Inquiry:**

- _ceim donavi ta libron?_ → **"To whom are you giving the book?"**
- _ceon donavi meim?_ → **"What are you giving to me?"**

- [x] **Specific Selection:**

- _cualon libron donavi meim?_ → **"Which book are you giving to me?"**

- [x] **Temporal Reference:**

- _cuando donovi ta libron meim?_ → **"When are you going to give the book to me?"**
- **Explanation:** Implies it is **in the future**.

- [x] **Quantity Inquiry:**

- _cuót libron donavi meim?_ → **"How many books are you giving to me?"**

---

### **Asking for the Subject of the Verb**

- [x] **Keeps only the known parts**, ensuring clarity in reference.

| **Sentence**                    | **Translation**                                    | **Explanation**                                                   |
| ------------------------------- | -------------------------------------------------- | ----------------------------------------------------------------- |
| _ceis donav ta libron meim?_    | "Who is giving the book to me?"                    | No indication about **who or how many** are involved.             |
| _cesis donaves ta libron meim?_ | "Who is giving the book to me?"                    | **Third-person plural**—multiple people, **excluding you or me**. |
| _cesis donavis ta libron meim?_ | "Who among you (plural) is giving the book to me?" | Used as a **follow-up question** when the subject is unclear.     |

## **Common Questions**

- [x] _cei seve de ve namis?_ → **"What's your name?"**
- [x] _cume savi?_ → **"How are you?"**
- [x] _ex cove sevi?_ → **"Where are you from?"**
