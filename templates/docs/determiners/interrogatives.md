# <x-trans>Questions</x-trans>

<x-trans>Questions in Arcadia are formed using a **question particle**, followed by an affirmative sentence **with the unknown part omitted**.</x-trans>

- [x] **<x-trans>Yes/no questions</x-trans>** →<x-trans> Indicated by **rising intonation** in spoken Arcadia.</x-trans>
- [x] **<x-trans>Open-ended questions</x-trans>** → <x-trans>Use **interrogative particles**, which combine with **prepositions and declensions** as expected.</x-trans>

## **<x-trans>Question Particles</x-trans>**

| **<x-trans>Particle</x-trans>** | **<x-trans>English</x-trans>**       | **<x-trans>Meaning</x-trans>**      |
| ------------------------------- | ------------------------------------ | ----------------------------------- |
| **cu**                          |                                      | <x-trans>Yes/no question</x-trans>  |
| **ce**                          | <x-trans>Who/what</x-trans>          | <x-trans>Generic inquiry</x-trans>  |
| **cuál**                        | <x-trans>Which</x-trans>             | <x-trans>Choice selection</x-trans> |
| **cove**                        | <x-trans>Where</x-trans>             | <x-trans>Location-based</x-trans>   |
| **cuando**                      | <x-trans>When</x-trans>              | <x-trans>Time reference</x-trans>   |
| **cuót**                        | <x-trans>How much/how many</x-trans> | <x-trans>Quantity inquiry</x-trans> |
| **cume**                        | <x-trans>How</x-trans>               | <x-trans>Manner inquiry</x-trans>   |

- [x] <x-trans>**Interrogative words can combine with prepositions**, allowing **expanded meanings**:</x-trans>

    - _pro ce_ → "<x-trans>Why</x-trans>"
    - _por ce_ → "<x-trans>For whose benefit</x-trans>"

!!! note "<x-trans>Handling Unknowns</x-trans>"

    <x-trans>In Arcadia, asking a question **causes affected suffixes to drop**:</x-trans>

        - <x-trans>**"&{cuót}"** removes **quantity markers**, simplifying nouns.</x-trans>
        - <x-trans>**"&{cuando}"** drops **time suffixes** when asking about duration.</x-trans>
        - <x-trans>**"&{ce}"** removes **person suffixes** when querying the subject of a verb.</x-trans>

---

## **<x-trans>Examples</x-trans>**

- [x] **<x-trans>Yes/No Question</x-trans>:**

    - _cu donavi ta libron meim?_ → **"<x-trans>Are you giving the book to me?</x-trans>"**

- [x] **<x-trans>Method Inquiry</x-trans>:**

    - _cume donavi ta libron meim?_ → **"<x-trans>How are you giving the book to me?</x-trans>"**
        - **<x-trans>Explanation</x-trans>:** <x-trans>Asking about the method—e.g., by hand, by post.</x-trans>

- [x] **<x-trans>Location Inquiry</x-trans>:**

- _cove donavi ta libron meim?_ → **"<x-trans>Where are you giving the book to me?</x-trans>"**

- [x] **<x-trans>Subject Inquiry</x-trans>:**

- _ceim donavi ta libron?_ → **"<x-trans>To whom are you giving the book?</x-trans>"**
- _ceon donavi meim?_ → **"<x-trans>What are you giving to me?</x-trans>"**

- [x] **<x-trans>Specific Selection</x-trans>:**

- _cualon libron donavi meim?_ → **"<x-trans>Which book are you giving to me?</x-trans>"**

- [x] **<x-trans>Temporal Reference</x-trans>:**

- _cuando donovi ta libron meim?_ → **"<x-trans>When are you going to give the book to me?</x-trans>"**
- **<x-trans>Explanation</x-trans>:** <x-trans>Implies it is **in the future**.</x-trans>

- [x] **<x-trans>Quantity Inquiry</x-trans>:**

- _cuót libron donavi meim?_ → **"<x-trans>How many books are you giving to me?</x-trans>"**

---

### **<x-trans>Asking for the Subject of the Verb</x-trans>**

- [x] <x-trans>**Keeps only the known parts**, ensuring clarity in reference.</x-trans>

| **<x-trans>Sentence</x-trans>** | **<x-trans>Translation</x-trans>**                                    | **<x-trans>Explanation</x-trans>**                                                   |
| ------------------------------- | --------------------------------------------------------------------- | ------------------------------------------------------------------------------------ |
| _ceis donav ta libron meim?_    | "<x-trans>Who is giving the book to me?</x-trans>"                    | <x-trans>No indication about **who or how many** are involved.</x-trans>             |
| _cesis donaves ta libron meim?_ | "<x-trans>Who is giving the book to me?</x-trans>"                    | <x-trans>**Third-person plural**—multiple people, **excluding you or me**.</x-trans> |
| _cesis donavis ta libron meim?_ | "<x-trans>Who among you (plural) is giving the book to me?</x-trans>" | <x-trans>Used as a **follow-up question** when the subject is unclear.</x-trans>     |

## **<x-trans>Common Questions</x-trans>**

- [x] _cei seve de ve namis?_ → **"<x-trans>What's your name?</x-trans>"**
- [x] _cume savi?_ → **"<x-trans>How are you?</x-trans>"**
- [x] _ex cove sevi?_ → **"<x-trans>Where are you from?</x-trans>"**
