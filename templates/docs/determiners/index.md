# <x-trans>Determiners</x-trans>

<x-trans>Determiners are words that precede a noun, providing additional information about **its reference, quantity, or specificity**.</x-trans>

<x-trans>Arcadia recognizes **six types** of determiners:</x-trans>

- **[<x-trans>Articles</x-trans>][article]** → <x-trans>Defines specificity:</x-trans> <x-trans>_the_ tree, _an_ apple.</x-trans>
- **[<x-trans>Demonstrative determiners</x-trans>][demonstrative determiner]** → <x-trans>Indicates proximity:</x-trans> <x-trans>_this_ machine, _that_ person.</x-trans>
- **[<x-trans>Interrogative determiners</x-trans>][interrogative determiner]** → <x-trans>Forms questions:</x-trans> <x-trans>_which_ book, _what_ blanket.</x-trans>
- **[<x-trans>Possessives</x-trans>][possessive]** → <x-trans>Expresses ownership:</x-trans> <x-trans>_Maria's_ bag, _my_ bed.</x-trans>
- **[<x-trans>Quantifiers</x-trans>][quantifier]** → <x-trans>Describes quantity:</x-trans> <x-trans>_a few_ dogs, _each_ child.</x-trans>
- **[<x-trans>Numerals</x-trans>][numeral]** → <x-trans>Specifies an exact number:</x-trans> <x-trans>_two_ dolphins, _a million_ dollars.</x-trans>

- [x] <x-trans>**Determiners can be combined**, as in **"the three musketeers."**</x-trans>

### <x-trans>**Determiner Rules in Arcadia**</x-trans>

- [x] <x-trans>**A determiner is mandatory** unless the noun is in the **zero**, or **collective number**, where **no determiner normally used**.</x-trans>
- [x] <x-trans>**Proper nouns** do **not** require a determiner but **can take one** when specificity needs to be emphasized.</x-trans>
