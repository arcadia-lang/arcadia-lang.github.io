# <x-trans>Quantifiers</x-trans>

<x-trans>Quantifiers are **determiners** that indicate the **quantity of a noun**.</x-trans>

- [x] **<x-trans>Nullar and collective numbers</x-trans>** → <x-trans>Naturally make **"all" and "none" redundant**, simplifying expressions.</x-trans>
- [x] **<x-trans>Usage Restrictions</x-trans>** → <x-trans>**Quantifiers cannot be used with uncountable nouns**, since the **number already conveys quantity**.</x-trans>

### **<x-trans>Core Quantifiers</x-trans>**

<x-trans>Arcadia defines the following **quantifiers**:</x-trans>

- **_poco_** → **"<x-trans>A few</x-trans>"** (<x-trans>small but distinct amount</x-trans>).
- **_aliame_** → **"<x-trans>Some</x-trans>"** (<x-trans>can refer to one or multiple</x-trans>).
- **_oliame_** → **"<x-trans>Some</x-trans>"** (<x-trans>cannot refer to one, only plural</x-trans>).
- **_multe_** → **"<x-trans>Many</x-trans>"** (<x-trans>large quantity</x-trans>).
- **_cisca_** → **"<x-trans>Each</x-trans>"** (<x-trans>individual emphasis</x-trans>).

### **<x-trans>Usage Notes</x-trans>**

- [x] **"aliame"** → <x-trans>Even though it **can refer to one**, it always **takes a plural noun form**.</x-trans>
- [x] **"cisca"** →
    - <x-trans>Used with **singular nouns** to highlight **individual entities**.</x-trans>
    - <x-trans>Used with **collective nouns** to emphasize **both individuals and the group**.</x-trans>

---

## **<x-trans>Examples</x-trans>**

- [x] **<x-trans>Collective Focus (`&{puereris}` in collective form)</x-trans>**

    - _edravur puereris e pomon_ → **"<x-trans>Every child is eating an apple.</x-trans>"**
    - **<x-trans>Explanation</x-trans>:** <x-trans>The **collective noun** and verb highlight **the group as a whole**.</x-trans>

- [x] **<x-trans>Singular Emphasis (`cisca` with singular noun)</x-trans>**

    - _edrave cisca pueris e pomon_ → **"<x-trans>Each child is eating an apple.</x-trans>"**
    - **<x-trans>Explanation</x-trans>:** <x-trans>The verb and noun remain **singular**, focusing on **individual children**.</x-trans>

- [x] **Hybrid Emphasis (`cisca` with collective noun)**

    - _edravur cisca puereris e pomon_ → **"<x-trans>Each and every child is eating an apple.</x-trans>"**
    - **<x-trans>Explanation</x-trans>:** <x-trans>The **collective noun + cisca** highlights **both individuals and the group**.</x-trans>
