# <x-trans>Numerals</x-trans>

<x-trans>Arcadia uses **two parallel number systems**:</x-trans>

- [x] **<x-trans>Decimal system</x-trans>** → <x-trans>Commonly used for general counting and calculations.</x-trans>
- [x] **<x-trans>Hexadecimal system</x-trans>** → <x-trans>Included for **future-proofing**, allowing expanded numerical representation.</x-trans>

### **<x-trans>Numbers and Writing</x-trans>**

- [x] <x-trans>**Hexadecimal notation uses `0-9` and `A-F`**, ensuring **no overlap with written words**, since Arcadia uses only **lowercase letters** for textual representation.</x-trans>
- [x] <x-trans>**Numbers are grouped using underscores (`_`)** for clarity and **unambiguous readability**.</x-trans>

---

## **<x-trans>Larger Numbers</x-trans>**

- [x] <x-trans>After **999_999**, both systems use **powers of 1_000_000** to create **names for arbitrarily large numbers**.</x-trans>
- [x] <x-trans>For **one-word power formations**, Arcadia **replaces "ad X potente"** with the **suffix "-ente"**.</x-trans>

### **<x-trans>Decimal Power Names</x-trans>**

| **<x-trans>Decimal</x-trans>** | **<x-trans>Name</x-trans>**                  |
| ------------------------------ | -------------------------------------------- |
| 1_000_000                      | **milonu**                                   |
| 1_000_000_000_000              | **dosente milonu** _(milonu ad dos potente)_ |

### **<x-trans>Hexadecimal Power Names</x-trans>**

| **<x-trans>Hexadecimal</x-trans>** | **<x-trans>Name</x-trans>**            |
| ---------------------------------- | -------------------------------------- |
| 0x1_000_000                        | **bet**                                |
| 0x1_000_000_000_000                | **dosente bet** _(bet ad dos potente)_ |
