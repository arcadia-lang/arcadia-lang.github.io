# <x-trans>Possessives</x-trans>

<x-trans>Arcadia uses the prepositions **"de"** and **"du"** to indicate possession.</x-trans>

- [x] <x-trans>**Possessives function as determiners**, appearing **before the noun**.</x-trans>
- [x] **<x-trans>Possessor-first order</x-trans>** → <x-trans>The **possessor always precedes the possessed** in phrase construction.</x-trans>

### **<x-trans>Alienable vs. Inalienable Possession</x-trans>**

- [x] **"de"** → <x-trans>Marks **inalienable possession**, where the possessor is **an integral part** of the possessed.</x-trans>
- [x] **"du"** → <x-trans>Marks **alienable possession**, where the possessor is **not inherently connected** to the possessed.</x-trans>

- [x] **<x-trans>No possessive pronouns</x-trans>** → <x-trans>Instead, possession is **expressed with "de" or "du" + pronoun**.</x-trans>

---

## **<x-trans>Examples</x-trans>**

- [x] **<x-trans>Alienable Possession</x-trans> (`du`)**

    - _save du me libris magne_ → **"<x-trans>My book is big.</x-trans>"**
    - **<x-trans>Explanation</x-trans>:** <x-trans>The book **is not inherently a part** of the speaker.</x-trans>

- [x] **<x-trans>Inalienable Possession</x-trans> (`de`)**

    - _seve de me de file viparenis adamo_ → **"<x-trans>The father of my child is Adam.</x-trans>"**
    - **<x-trans>Explanation</x-trans>:** <x-trans>The child's relationship to the father **is inherent**.</x-trans>

- [x] **<x-trans>Possessive Chaining</x-trans>**
    - _save ta libris du me_ → **"<x-trans>The book is mine.</x-trans>"**
    - **<x-trans>Explanation</x-trans>:** <x-trans>Ownership chain follows **possessor-first ordering**.</x-trans>
