## **<x-trans>Modifiers</x-trans>**

<x-trans>Arcadia includes **two adjectives** for **shade**:</x-trans>

| <x-trans>Arcadia</x-trans> | <x-trans>English</x-trans> |
| -------------------------- | -------------------------- |
| lumo                       | <x-trans>light</x-trans>   |
| scuro                      | <x-trans>dark</x-trans>    |

<x-trans>and **four adjectives** for **color finish**:</x-trans>

| <x-trans>Arcadia</x-trans> | <x-trans>English</x-trans>  |
| -------------------------- | --------------------------- |
| metallico                  | <x-trans>metallic</x-trans> |
| matto                      | <x-trans>matte</x-trans>    |
| gloso                      | <x-trans>glossy</x-trans>   |
| neono                      | <x-trans>neon</x-trans>     |

<x-trans>Shade always **precedes** the color finish.</x-trans>

## **<x-trans>Examples</x-trans>**

1. _azulis glosis_

    - **translation:** glossy blue

2. _sangis scuris_

    - **translation:** dark red

3. _verdis lumis mattis_

    - **translation:** light matte green
