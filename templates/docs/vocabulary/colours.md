## <<Modifiers>>

Arcadia includes **two adjectives** for **shade**:

| Arcadia | English |
| ------- | ------- |
| lumo    | light   |
| scuro   | dark    |

and **four adjectives** for **color finish**:

| Arcadia   | English  |
| --------- | -------- |
| metallico | metallic |
| matto     | matte    |
| gloso     | glossy   |
| neono     | neon     |

Shade always **precedes** the color finish.

## <<Examples>>

1. _azulis glosis_

   - **translation:** glossy blue

2. _sangis scuris_

   - **translation:** dark red

3. _verdis lumis mattis_
   - **translation:** light matte green
