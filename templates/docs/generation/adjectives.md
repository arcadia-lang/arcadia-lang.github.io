# ((Adjectives))

Arcadia has a smaller set of adjectives than the other languages, but it has a rich generative system.

## ((Antonyms))

First, we will explore how we can create antonyms in Arcadia.

### _mal_

One of the most common prefixes in Arcadia is _mal_, which means "bad".
It can be used to change the meaning of an adjective to its opposite, in cases that the pair of antonyms can be associated with a positive/negative connotation.
For example, _boni_ (good) can be changed to _malboni_ (bad), but _novi_ (new) cannot be changed to _malnovi_ (old), as the old/new distinction is not based on a positive/negative connotation.

Adjectives that can use the _mal_ prefix include:

- boni: good
- veri: true
