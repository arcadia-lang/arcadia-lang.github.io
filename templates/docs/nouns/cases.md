# Cases

Arcadia has **seven grammatical cases**, each marked by a **suffix**.
These cases define the **syntactic role** of a noun within a sentence.

We will use **"pan"** (_bread_) as our example noun.

## **Case System**

- **Nominative** (**"-is"**) → Marks the **subject** of the sentence: _panis_
- **Accusative** (**"-on"**) → Marks the **direct object**: _panon_
- **Dative** (**"-im"**) → Marks the **indirect object** (recipient): _panim_
- **Ablative** (**"-a"**) → Marks the **indirect object** (donor/source): _pana_
- **Predicative** (**"-i"**) → Used with **linking verbs** to express **attributes**: _pani_
- **Adpositional** (**"-e"**) → Used after **prepositions**: _pane_
- **Vocative** (**"-o"**) → Used for **direct address**: _pano_

## **Vowel Merging Rules**

If a noun **ends in a back vowel** (_&{a}, &{o}, &{u}_), the suffix merges with the vowel:

- [x] **Nominative & Accusative:** The **suffix vowel is dropped** (_instead of "luna-is" → "lunas"_).
- [x] **Other cases:** The **noun's final vowel is dropped** (_instead of "luna-on" → "lunon"_).
