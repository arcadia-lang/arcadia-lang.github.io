# Cases

Arcadia has nine grammatical cases, each marked by a suffix.
These cases define the syntactic role of a noun within a sentence.

## Case Suffixes

The suffixes for each case are as follows:

- Nominative: "-is" →
  - Marks the **focus** of the sentence

- Ergative: "-er" →
  - Marks the **unfocused actor** of action verbs

- Absolutive: "-il" →
  - Marks the **unfocused experiencer** of experience verbs

- Accusative: "-an" →
  - Marks the **unfocused patient** of action verbs

- Dative: "-al" →
  - Marks the **unfocused receiver** of action verbs

- Thematic: "-ath" →
  - Marks the **unfocused stimulus** of experience verbs

- Predicative: "-i" →
  - Marks the **predicate** of a linking verb

- Prepositional: "-e" →
  - Used after **prepositions**

- Vocative: "-o" →
  - Used for **direct address**

- Instrumental: "-a" →
  - Marks the **instrument** of an action

- Benefactive: "-u" →
  - Marks the **beneficiary** of an action

## Vowel Merging Rules

In Arcadia, nouns end in a vowel, and the case suffixes also start with a vowel.
Front vowels (_i_ and _e_) and consonants (in other numbers apart from singular) do not merge with the suffix vowel.
If a noun **ends in a back vowel** (_a, o, u_), the suffix merges with the vowel.
In the consonant cases, the suffix vowel is dropped, as the consonant offers enough distinction.
But in the other cases, the noun's final vowel is dropped.
